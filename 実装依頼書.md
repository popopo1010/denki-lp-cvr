## 目的
CVR改善のため、既存LPに以下の要素を追加する

## 追加ファイル（2ファイル）
- `cvr-boost.css` … 追加スタイル
- `cvr-boost.js` … 通知ローテーション・離脱防止・GTMイベント送信

※既存のHTML構造・CSS・JSには変更を加えません。追加のみです。

---

## 実装方法（いずれかで対応）

### A案：テーマファイルに追加（推奨）

1. 以下にファイルをアップロード
   - `wp-content/themes/original-thema/assets/css/cvr-boost.css`
   - `wp-content/themes/original-thema/assets/js/cvr-boost.js`

2. 該当ページのテンプレート（またはfunctions.php）で読み込み追加
   ```html
   <!-- </head> の直前に追加 -->
   <link rel="stylesheet" href="/wp-content/themes/original-thema/assets/css/cvr-boost.css">
   ```
   ```html
   <!-- </body> の直前に追加 -->
   <script src="/wp-content/themes/original-thema/assets/js/cvr-boost.js" defer></script>
   ```

### B案：GTM（Google Tag Manager）で注入

GTM（GTM-KV525PZ）にて、カスタムHTMLタグで読み込む方法でもOKです。
