# 独自予約システム（TimeRex不要）

LPの電話・名前を使い、**サンクスで日時だけ選ぶ** → **Googleカレンダーに予定作成** → **スプシ更新 + Slack通知**。

## 構成

```
サンクス（独自UI・3日表示）
  ↓ JSONP 空き枠取得
GAS + Googleカレンダー（担当者の空き）
  ↓ 予約 POST
Googleカレンダーにイベント作成 + スプシ + Slack
```

### Googleカレンダー（予約は自動であなたのカレンダーに入る）

LPで枠を選んで「予約する」と、**GASデプロイアカウントのGoogleカレンダー**に予定が1件作成されます（追加操作不要）。

1. GASエディタで **`getBookingCalendarInfo`** を実行 → ログの `calendar_id` をコピー
2. プロジェクトの設定 → **スクリプトプロパティ** → `BOOKING_CALENDAR_ID` = その ID（例: `yuki.shibayama@xchange-inc.com`）
3. 未設定の場合は **デプロイユーザー（yuki.shibayama@xchange-inc.com）のメインカレンダー** に入ります
4. 手動で ID を確認: Googleカレンダー → 対象カレンダー ⚙ → **カレンダーの統合** → **カレンダーID**
5. TimeRex用 `yuki.shibayama_34d4` とは別。独自予約の正本は **このGoogleカレンダー**

## 2. 受付ルール（GASで自動）

| 項目 | 既定値 | プロパティで変更 |
|------|--------|------------------|
| 枠の長さ | 15分 | `BOOKING_SLOT_MINUTES` |
| 受付時間 | 平日 9:00〜18:00 | `BOOKING_START_HOUR` / `BOOKING_END_HOUR` |
| 何時間後から | 2時間後 | `BOOKING_LEAD_HOURS` |
| 何日先まで | 14日 | `BOOKING_DAYS_AHEAD` |

空き判定は **そのカレンダーの既存予定と重ならない枠** のみ表示します。

## 3. テスト

1. LP送信 → サンクス（`thanks/preview.html` でも可）
2. 3日分の時間ボタンが表示される
3. 時間を選び「この日時で予約する」
4. **Googleカレンダー（BOOKING_CALENDAR_ID で指定したカレンダー）**に予定が入る
5. 入らない場合: GASで `testBookSlot` を実行 → 同じカレンダーにテスト予定が入るか確認
5. Slackに日時が届く（Webhook設定時）

## 4. TimeRexとの比較

| | 独自 | TimeRex |
|--|------|---------|
| 月額 | なし（GAS無料枠） | あり |
| 名前メール再入力 | なし | 削除不可 |
| 3日表示 | UIで制御 | 埋め込み依存 |
| 空きの元 | 自社Googleカレンダー | TimeRex設定 |
