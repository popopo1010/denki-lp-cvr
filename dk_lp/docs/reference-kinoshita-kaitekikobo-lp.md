## 4. ストーリーLP（`/concept/`）の構造分解

| 要素 | 内容 | 当社への転用 |
|------|------|-------------|
| 書き出し | 「試しては辞め…」共感 | エージェント不信版に置換 |
| 差別化 | モニター・類似成分比較 | 口コミ・非公開求人の選び方 |
| 反流行 | 新成分ビジネス批判 | 求人量産型批判（慎重に） |
| 少SKU | 品質のため商品数少 | 工種特化・非公開に絞る理由 |
| トーン | 明朝・余白・医療寄り信頼 | 当社は青系ビジネス信頼 |
