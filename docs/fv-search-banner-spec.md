# FVバナー仕様書 — denkikouji-search（検索サイト訴求）

実写真バナー作成用の仕様。現行の暫定SVG（`assets/img/first_banner_search.svg`）の
レイアウトを、電気現場の実写真に置き換えるための指示書。

対象LP: `https://denkilp.builders-job.com/denki-lp-cvr/denkikouji-search/`
（ABテスト用・FVのみが現行 `denkikouji-v2` との差分）

## 0. コンセプト（木下基準）
- **検索サイトだと3秒で伝わる**こと（紹介＝受け身ではなく、自分で探せる）
- LP本体が明るいトーンなので、**バナーも明るく・高コントラスト**で馴染ませる
- **権威・数値はバナーに入れない**（直下の信頼バー＝厚労省許可/SSL/完全無料、数値＝13,829/34,513/94% が担当。重複は情報過多で逆効果）

## 1. サイズ・書き出し
- アスペクト比: **約 2.5:1（横長・短め）**
- 推奨書き出し: **1600 × 640px**（Retina対応で 2x = 3200 × 1280px）
- 形式: **WebP（主）＋ JPG（フォールバック）** の2種
- ファイル名（厳守）:
  - `first_banner_search.webp`
  - `first_banner_search.jpg`
- モバイルでは横幅100%で表示（実高さ ≈ 135〜170px）。**文字は大きめ**に。

## 2. レイアウト（左寄せ・現行SVG踏襲）
```
┌───────────────────────────────────────────────┐
│ [電気業界 特化]（緑ピル）                          │  ← 左上
│                                                 │
│  電気業界の求人を、        （濃緑・極太の見出し）    │
│  ┌─────────────────────────────┐ 　            │
│  │ あなたの条件で検索        🔍 │（白い検索ボックス）│
│  └─────────────────────────────┘               │
│  資格・年収・勤務地で絞り込み （サブ・グレー）        │
└───────────────────────────────────────────────┘
        背景＝電気工事の現場写真（明るめ）
```
- **検索ボックスは白角丸＋緑枠**、右端に**緑の丸い検索ボタン（白い虫めがね）**。検索サイト感の要。
- テキストは**左半分に集約**。背景写真は右側に人物/現場が来る構図がベター。
- 写真の左側は**白〜薄いオーバーレイ**を敷き、検索ボックス・見出しの可読性を確保。

## 3. コピー（そのまま使用・改変不可の確定文言）
| 要素 | 文言 |
|------|------|
| 専門ピル | 電気業界 特化 |
| 見出し | 電気業界の求人を、 |
| 検索ボックス内 | あなたの条件で検索 |
| サブ | 資格・年収・勤務地で絞り込み |

| 検索ボックス地 | `#ffffff` |

## 5. 入れてはいけない要素（重複・情報過多の回避）
- 利用満足度 94%（直下の数値帯にあり）
- 厚生労働大臣許可 / 許可番号（直下の信頼バーにあり）
- 完全無料 / SSL / 利用者数（同上）
- 「オススメの求人を紹介」等の**受け身・紹介トーン**（検索＝能動に統一）

## 6. 反映方法
1. `first_banner_search.webp` / `.jpg` を作成して共有
2. こちらで `assets/img/` に配置し、`denkikouji-search/index.html` のFV参照を
   SVG→WebP/JPG（`<picture><source webp><img jpg>`）に切替＋preload更新
3. `main` push → 自動デプロイ → ハードリロードで反映

## 7. 注意
- 94%などの数値を将来バナーに復活させる場合は**根拠確定が前提**（木下「98%はボツ」精神＝検証なき強い主張は不可）。
