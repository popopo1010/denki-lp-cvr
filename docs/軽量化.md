# 軽量化・高速化（実装済み一覧）

## サンクス `thanks-v2`

| 施策 | 効果 |
|------|------|
| LandingHub 削除 | 外部JS 2本削減 |
| WP `style.css`（52KB）削除 → `thanks-page.css` | CSS 大幅削減 |
| `cvr-boost-v2.css`（26KB）削除 | 同上 |
| GTM 遅延読み込み（操作 or 6秒） | 初回描画をブロックしない |
| `thanks-booking-bootstrap.js` 1本化 | HTTP 往復 −1 |
| `booking-slots.json` preload | 枠表示を前倒し |
| 口コミ `content-visibility` | 下段の描画コスト削減 |
| deploy 時 minify | 転送量削減 |

## 空き枠

| 施策 | 内容 |
|------|------|
| 静的 JSON | CDN から即表示 |
| GAS CacheService | 3分キャッシュ |
| 5分 sync workflow | JSON 更新 |
| LP 送信時プリウォーム | 遷移1秒中に取得 |
| 取得日数 5日 | GAS 負荷軽減 |

## LP（`*-v2`）

| 施策 | 内容 |
|------|------|
| WP CSS 非同期 | `media="print" onload` |
| Critical CSS インライン | FV 即描画 |
| `app-v2.js` defer | HTML パース優先 |
| GTM lazy | 操作後読込 |
| サンクス遷移 1.5秒→1秒 | 待ち短縮 |

## deploy（`.github/workflows/deploy.yml`）

- `app-v2.js` / 予約JS / GTM / 主要 CSS を esbuild minify
- `thanks-v2` と全 `*-v2/index.html` を HTML minify
