## 空き枠

| 施策 | 内容 |
|------|------|
| 静的 JSON | CDN から即表示 |
| GAS CacheService | 3分キャッシュ |
| 5分 sync workflow | JSON 更新 |
| LP 送信時＋電話入力(step06)でプリウォーム | サンクス到達前に枠取得 |
| localStorage 30分キャッシュ | セッション切れても即表示 |
| サンクスで予約JSをカレンダー直後に実行 | defer待ちを回避 |
| スケルトン即表示 | 読み込み中の空白を防止 |
| サンクス遷移 600ms | 1秒→短縮 |
| 取得日数 5日 | GAS 負荷軽減 |

## LP（`*-v2`）

| 施策 | 内容 |
|------|------|
| WP CSS 非同期 | `media="print" onload` |
| Critical CSS インライン | FV 即描画 |
| `app-v2.js` defer | HTML パース優先 |
| GTM lazy | 操作後読込 |
| サンクス遷移 1.5秒→1秒 | 待ち短縮 |

- `thanks-v2` と全 `*-v2/index.html` を HTML minify
