## 3. 肌潤風呂LP（`/hb/`）の構造分解

広告クリック後の**セールスレター型** + ページ下部に**申込フォーム一体型**。

| 順 | ブロック | 内容 | 木下手法との対応 |
|----|----------|------|------------------|
| 1 | FV | 悩みフック（入浴で乾燥）→ ベネフィット（1日中しっとり）→ 商品名 | #5 誰に何を、#25 コピー順序 |
| 2 | 安心バッジ | 無添加系3点（着色料・香料・パラベン等） | #13 買わない理由を潰す |
| 3 | 権威 | 女性誌掲載コラージュ | 社会的証明（当社は★4.8・許可） |
| 4 | 数値証拠 | **89%** が使い続けたい（円グラフ） | 要確認数値の扱い（CLAUDE.md） |
| 5 | 中間CTA | スタッフ顔写真＋「お客様の声へ」 | エモーションリレー #26 |
| 6 | 以降 | お客様の声・成分・比較・FAQ（スクロール） | ロングLP定石 |
| 7 | 申込 | 定期コース3種・個数で割引・解約条件明示 | #27–28 マイクロコピー、#55 利益設計 |

**申込フォームで見えるマイクロコピー例**（2026-06-04 時点）

- コース: お試し定期 / 3ヵ月毎 / 年間購入 — 価格差と「◯回受け取り後解約可」を併記
- 単品9ヵ月 vs 定期の**差額**を数字で見せる（損失回避）
- 「入力お疲れ様でした！下のボタンをクリックしてご注文完了」 — ゴール直前の労い

## 4. ストーリーLP（`/concept/`）の構造分解

購入直前ではなく、**信頼醸成・ブランド理解**用の長文。

| 要素 | 内容 | 当社への転用 |
|------|------|-------------|
| 書き出し | 「試しては辞め…満足できないから」共感 | 転職LPの「エージェント不信」版に置換 |
| 差別化 | モニター調査・類似成分比較・驚きの良さだけ商品化 | → 口コミ・非公開求人の**選び方**の説明 |
| 反流行 | 新成分ビジネスへの批判 | → 求人サイト量産型への批判（慎重に） |
| 少SKU | 商品数が少ない理由＝品質 | → 工種特化・非公開に絞る理由 |
| トーン | 明朝・余白・蝶の装飾 — 医療寄り信頼 | 当社は青系ビジネス信頼（thanks-v2） |
